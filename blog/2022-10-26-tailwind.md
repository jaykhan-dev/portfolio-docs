## Setup

### Install commands

```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

### Configure `tailwind.config.js`

```js
// eslint-disable-next-line no-undef
module.exports = {
  content: ["./index.html", "./src/**/*.{vue,js,ts,jsx,tsx}"],
  theme: {
    extend: {},
  },
  plugins: [],
  darkMode: "class",
  variants: {
    extend: { display: ["dark"] },
  },
};
```

### `index.css`

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```

### `main.js`

```js
import { createApp } from "vue";
import App from "./App.vue";
import "./index.css";

createApp(App).mount("#app");
```

### `postcss.config.js`

```js
/* eslint-disable no-undef */
module.exports = {
  plugins: {
    tailwindcss: {},
    autoprefixer: {},
  },
};
```

### run application

```bash
npm run dev
```
